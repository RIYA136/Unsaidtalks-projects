
# 📊 Finance Agent

Hey! 👋 This is a simple Python script that helps you understand how companies like Apple, Tesla, or NVIDIA are doing in the stock market — using AI and real-time data.

It gives you:
- ✅ Latest stock price and key stats (like P/E ratio, EPS)
- 📰 Company news + analyst opinions
- 📈 Easy summaries with tables and emojis

---

## 🚀 How to Use

1. Install required packages:

```bash
pip install agno openai
```

2. Set your OpenAI API key:

```bash
export OPENAI_API_KEY="your_api_key"  # Mac/Linux
```

```cmd
set OPENAI_API_KEY=your_api_key       # Windows
```

3. Run the script:

```bash
python finance_agent_intern_version.py
```
